# Automation & The End of Tier 1

## 1. Orientation

### What this module covers
This module covers the **Evolution of the Analyst**.

### Fit in the Learning Path
Tier 1 (Ticket closing robots) is disappearing. Automation does it better.

### Learning Objectives
By the end of this module, you should understand:
*   **SOAR**.
*   **The Rise of the Engineer-Analyst**.
*   **Career Logic**.

---

## 2. Core Content

### The Tier 1 Death
*   *Old Job*: Read alert. Check IP intelligence. Close false positive. (Repeat 100x).
*   *New Job*: Write a python script to do that.
*   *Implication*: Entry-level jobs require coding skills now.

### SOAR (Security Orchestration, Automation and Response)
*   **Playbook**: Phishing Email -> Extract URL -> Scan in VirusTotal -> If Bad, Block on Firewall -> Delete Email -> Notify User.
*   All done in 3 seconds without human intervention.

---

## 3. Guided Practice

### The Pivot
**You are**: A Tier 1 Analyst.
**The Threat**: Automation is coming.
**Action**:
1.  Learn Python / PowerShell.
2.  Ask to build the playbook for the ticket you hate the most.
3.  Become the **Automation Engineer**.

---

## 4. Reflection Check

1.  **Quality**: Automating a bad process just creates bad results faster. Optimize first.
2.  **Oversight**: Who watches the bots? (Tier 2/3).
3.  **Humanity**: You still need humans for *novel* attacks. AI creates efficiency, Humans provide creativity.

---

## 5. Completion Criteria

This module is complete when:
1.  You embrace **Automation**.
2.  You learn a **Scripting Language**.
3.  You understand that **Manual Ops** is a bug.
