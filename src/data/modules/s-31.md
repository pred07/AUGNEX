# Threat Intelligence Platforms (MISP, OpenCTI)

## 1. Orientation

### What this module covers
Where do you put all this data? Spreadsheets fail. We use **TIPs (Threat Intelligence Platforms)** to store, correlate, and share intelligence.

### Learning Objectives
*   Use **MISP** (Malware Information Sharing Platform).
*   Use **OpenCTI** (Knowledge Graph).
*   Understand **STIX/TAXII**.

---

## 2. Core Content

### MISP (The Standard)
*   **Events:** Containers for intel (e.g., "Event 101: Emotet Wave").
*   **Attributes:** IOCs inside events (IPs, hashes).
*   **Correlation:** MISP automatically links Event 101 to Event 50 because they share the same IP. *Magic.*
*   **Sharing:** Syncing your MISP with other companies (ISACs).

### OpenCTI (The Graph)
*   More visual. Uses the STIX 2 data model.
*   Great for visualizing Actor relationships (Diamond Model).

### STIX/TAXII
*   **STIX:** The Language (JSON format). "This is an IP."
*   **TAXII:** The Pipe (Transport). "Here is the STIX package."
