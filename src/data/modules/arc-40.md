# The Infinite Game

## 1. Orientation

### What this module covers
This module covers **The End**. Which is really the beginning.

### Fit in the Learning Path
There is no "Secure". There is only "Securing".

### Learning Objectives
By the end of this module, you should understand:
*   **Finite vs Infinite Games**.
*   **Burnout vs Purpose**.
*   **The Watch Continues**.

---

## 2. Core Content

### The Concept (Simon Sinek)
*   **Finite Game**: Football. Fixed rules. Clear winner. Ends in 90 mins.
*   **Infinite Game**: Business/Security. Rules change. New players enter. The goal is to keep playing.
*   *Mistake*: Trying to "Win" security. You can't.

### The Purpose
We protect the innovative capacity of humanity. We protect the hospital. We protect the bank. We protect the vote.
*   It is a noble profession.
