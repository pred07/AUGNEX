# Parameter Discovery and Mapping

## 1. Orientation

### What this module covers
This module goes deeper than endpoints. We look for hidden **Parameters**. E.g., `page.php?debug=true`.

### Fit in the Learning Path
Sometimes the page is visible, but the vulnerability is hidden behind a parameter that isn't documented. "Mass Assignment" and "Debug" bugs live here.

### Learning Objectives
By the end of this module, you should understand:
*   **Query Parameters** (`?id=1`).
*   **POST Body Parameters**.
*   Tools like `Arjun` or `ParamMiner`.

---

## 2. Core Content

### The Hidden Keys
Developers write code like:
```php
if ($_GET['test_mode'] == 'on') {
    enable_root();
}
```
If you visit `site.com`, nothing happens. If you visit `site.com?test_mode=on`, you become root.
How do you find `test_mode`? You fuzz it.

### Discovery Tools
*   **Arjun**: Specialized parameter discovery tool.
*   **Burp Param Miner**: Extension for Burp Suite.
*   **FFuF**: Can be used here too.

---

## 3. Guided Practice

### Fuzzing for Parameters
**Conceptual Command**:
```bash
ffuf -u http://site.com/page.php?FUZZ=1 -w parameters.txt
```
*   **Logic**: It replaces `FUZZ` with `id`, `user`, `admin`, `debug`, `source`...
*   **Success**: If the response *Size* or *Lines* differs from the baseline, you likely found a valid parameter.

**Common Targets**:
*   `debug=true`
*   `admin=true`
*   `redirect=url`

---

## 4. Reflection Check

1.  **Mass Assignment**: In modern APIs, if you send `{ "user": "me", "is_admin": true }` effectively guessing the `is_admin` parameter, does it work? (Often yes, in NodeJS/Rails apps).
2.  **Diffing**: Why is response size pertinent? (Because a valid parameter usually changes the page content, while an invalid one is ignored).
3.  **Hiding**: Where else can parameters hide? (Headers, Cookies).

---

## 5. Completion Criteria

This module is complete when:
1.  You understand that URLs are not just paths, but inputs.
2.  You can explain the concept of Parameter Fuzzing.
3.  You treat every page as a potential puzzle box waiting for the right key (Parameter).
