# Integrating Intel into SIEM/SOAR

## 1. Orientation

### What this module covers
The "Finish" line. Getting intel into the fight. SIEM (Security Information and Event Management) and SOAR (Orchestration).

### Learning Objectives
*   Ingest **Threat Feeds** into SIEM.
*   Write **Correlation Rules** using Intel.
*   Trigger **SOAR Playbooks** from Intel matches.

---

## 2. Core Content

### SIEM Integration
*   **Lookup Tables:** A CSV of bad IPs updated hourly.
*   **Query:** `index=firewall [ inputlookup bad_ips.csv | fields src_ip ]`.
*   *Theory:* Match *Live Traffic* against *Bad List*.

### SOAR Automation
*   **Trigger:** SIEM Alert ("Traffic to Bad IP").
*   **Playbook:**
    1.  Get IP Reputation (VirusTotal).
    2.  If Score > 5: Block on Firewall.
    3.  If Score < 5: Ask Analyst.
    4.  Ticket: Create JIRA.

---

## 3. Lab: Designing a Workflow
**Goal:** Automate Phishing Analysis.
1.  User forwards email to `phish@company.com`.
2.  SOAR extracts URLs.
3.  SOAR queries URLScan.io.
4.  If Malicious -> Delete email from all mailboxes (Purge).
5.  If Clean -> Email user "Safe".
